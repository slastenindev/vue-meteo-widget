# Changelog

All notable changes to this project will be documented in this file.

## [1.0.1] - 2026-05-10

### Fixed
- Added vite alias @.

## [1.0.0] - 2026-05-10

### Added
- Initial release of `vue-meteo-widget`.
- Vue 3 component `WeatherWidget` for displaying weather data.
- Open-Meteo API integration.
- `useWeather` and `useTime` composables for logic reuse.
- Support for dynamic icons (day/night themes).
- Local storage caching for weather data.
- Built-in TypeScript support with generated `.d.ts` files.
- Global and local installation support as a Vue plugin.
- Dark and Light themes support.
